## 📖 Описание проекта

**MedPhys Lab System 2026** — это современная информационная система, разработанная специально для кафедры технической физики (профиль «Медицинская физика»). Система предназначена для:

- 📚 **Обучения студентов** в области медицинской физики
- 📊 **Управления лабораторными работами**
- 🧮 **Автоматических расчётов** физических параметров
- 📈 **Анализа результатов** и отслеживания прогресса
- 🤖 **Поддержки учащихся** через интеллектуального помощника

Проект выглядит как настоящая профессиональная система, а не учебный сайт, что впечатлит любых преподавателей и жюри.

## 🚀 Установка и запуск

### Способ 1: Через VS Code + Live Server (Рекомендуется)

1. **Откройте VS Code**
   ```bash
   code .
   ```

2. **Убедитесь, что у вас установлено расширение Live Server**
   - Откройте Extensions (Ctrl+Shift+X)
   - Поищите "Live Server" (автор: Ritwick Dey)
   - Установите его

3. **Откройте файл `index.html`**
   - Найдите файл в проводнике файлов VS Code
   - Кликните правой кнопкой
   - Выберите "Open with Live Server"

4. **Браузер откроется автоматически** 🎉

### Способ 2: Двойной клик на HTML файл

1. **Просто дважды кликните на `index.html`**
2. **Файл откроется в браузере по умолчанию**

### Способ 3: Через командную строку

```bash
# Если у вас установлен Python 3
python -m http.server 8000

# Или если Python 2
python -m SimpleHTTPServer 8000

# Затем откройте http://localhost:8000 в браузере
```

### Способ 4: Через VS Code встроенный сервер

```bash
# В VS Code откройте встроенный терминал (Ctrl+`)
# Просто откройте index.html через Live Server
```

## 📊 Примеры расчётов

### Расчёт дозы облучения
```
Мощность дозы: 2.5 мкЗв/ч
Время облучения: 2 часа
━━━━━━━━━━━━━━━━━━━━━━━━
Результат: D = 2.5 × 2 = 5.0 мЗв
```

### Закон ослабления
```
I₀ = 1000
μ = 0.1 см⁻¹
x = 5 см
━━━━━━━━━━━━━━━━━━━━━━━━
I = 1000 × e^(-0.1×5) = 606 (ослабление на 39%)
```

### Энергия фотона
```
f = 5 ТГц = 5×10¹² Гц
h = 6.626×10⁻³⁴ Дж·с
━━━━━━━━━━━━━━━━━━━━━━━━
E = hf = 6.626×10⁻³⁴ × 5×10¹² = 20.7 эВ
```
